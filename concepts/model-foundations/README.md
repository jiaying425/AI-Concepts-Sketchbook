# AI Model Foundations

## Classic Deep Learning Models

The classic model sketch introduces:

- CNN: extracting patterns from visual data
- RNN: understanding sequences over time

- RL: learning from feedback and optimizing decisions

## Transformer Foundations

The transformer sketch explores:

- tokenization
- token embeddings
- positional encoding
- decoder-only transformer flow
- masked multi-head self-attention
- feedforward neural networks
- residual connections and layer normalization
- GPT-style parameter count intuition
